# Imbued

Imbued is a modular home‑server dashboard built with ASP.NET Razor Pages.  
It provides an easy to navigate, customizable interface for monitoring and managing your system, with support for draggable widgets and expandable modules.

## Features
- 🧩 Modular architecture (Media, Minecraft, Bluetooth, System, etc.)
- 📊 Customizable dashboard with draggable widgets
- ⚡ Fast, lightweight, and self‑hosted
- 🎨 Clean sidebar layout with responsive design
- 🔧 Built for extensibility and personal home‑server setups

## Roadmap
- Widget system with drag‑and‑drop + persistent layout
- System stats widgets (CPU, RAM, Disk, Network)
- Minecraft server integration
- Media server controls
- Bluetooth device management
- User settings + theme support
